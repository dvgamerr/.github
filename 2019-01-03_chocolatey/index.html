    วิธีติดตั้ง และ ใช้งาน Chocolatey

    Chocolatey recommend software for developer

    Published Thu, Jan 3, 2019 by Kananek Thongkam

    Estimated reading time: 1 min

    1. ติดตั้ง chocolatey ด้วย cmd as administrator และรันคำสั่งด้านล่าง
    @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
    1. ปิด terminal และเปิด cmd as administrator อีกครั้ง รันคำสั่งด้านล่างต่อ
    @echo off
choco feature enable -n allowGlobalConfirmation

choco install git.install --params "/NoGuiHereIntegratio /NoShellHereIntegration /NoShellIntegration" --force
